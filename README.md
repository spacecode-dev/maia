## Documentation
Visit [Documentation](https://spacecode.dev/cms-maia/docs) section in the website

## System Requirements
* PHP >= 7.2.28
* Composer
* Laravel Mix
* Node.js & NPM
* Laravel Framework 5.8.*
* Laravel Nova 2.9.4
* BCMath PHP Extension
* Ctype PHP Extension
* JSON PHP Extension
* Mbstring PHP Extension
* OpenSSL PHP Extension
* PDO PHP Extension
* Tokenizer PHP Extension
* XML PHP Extension
